## Environment Setup

Copy `.env.example` to `.env` in both `backend/` and `frontend/`:

**`backend/.env`:**
```
DATABASE_URL="postgresql://user:password@localhost:5432/todoflow_dev"
JWT_SECRET="your-secret-at-least-32-chars"
PORT=3001
NODE_ENV=development
```

**`frontend/.env.local`:**
```
NEXT_PUBLIC_API_URL=http://localhost:3001
```

## Development Commands

### Install dependencies (run once)
```bash
pnpm install --recursive
```

### Apply database migrations
```bash
cd backend && pnpm prisma migrate dev
```

### Start backend server
```bash
cd backend && pnpm dev
# Starts Hono on http://localhost:3001
```

### Start frontend server
```bash
cd frontend && pnpm dev
# Starts Next.js on http://localhost:3000
```

### Run all tests
```bash
cd backend && pnpm test          # Vitest + Supertest
cd frontend && pnpm test         # Vitest + React Testing Library
cd frontend && pnpm test:e2e     # Playwright (requires both servers running)
```

### Run quality gate
```bash
# From project root:
cd backend && pnpm lint && pnpm typecheck && pnpm test
cd frontend && pnpm lint && pnpm typecheck && pnpm test
```
