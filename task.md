## Group 8 ‚Äî V2 single-adapter migration

> Created 2026-04-07. Migrates the bot from the assumed "N market adapters per vault" model to the real Morpho Vault V2 model: **one adapter per protocol, N managed markets**. Every signature below has been verified against [`morpho-org/vault-v2`](https://github.com/morpho-org/vault-v2). Stories: A1, A2, B1, B2, E1, F1, F2.
>
> **Decisions** (carried from SPEC ¬ß Open Questions):
> - Vault model: 1 adapter + N managed markets
> - Demo curator role: deployer = owner = curator
> - Curator timelocks: leave at 0 at fresh deploy so the script can `multicall(submit + execute)` everything in one tx
> - Market discovery: configured at startup via `MANAGED_MARKETS_PATH` JSON
